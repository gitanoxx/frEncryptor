# frEncryptor 🛡️

**frEncryptor** es una herramienta avanzada de ofuscación y encriptación de código diseñada para proteger la propiedad intelectual de desarrolladores de **JavaScript** y **Python**. 

Desarrollado con un enfoque en la seguridad y la privacidad, frEncryptor permite transformar código legible en versiones protegidas, dificultando la ingeniería inversa sin comprometer la funcionalidad.

## ✨ Características

- **Soporte Multilenguaje:** Protección especializada para JavaScript (incluyendo TypeScript) y Python.
- **Modos de Protección:**
  - **Ofuscación:** Reestructuración de la lógica para hacerla ilegible.
  - **Encriptación:** Capas adicionales de seguridad para máxima protección.
- **Configuración Avanzada:**
  - Compactación de código.
  - Autodefensa contra formateo.
  - División de cadenas de texto.
  - Protección contra herramientas de depuración (Debug Protection).
- **Interfaz Moderna:** Diseño futurista con efectos visuales y experiencia de usuario fluida.
- **Privacidad Total:** Procesamiento en tiempo real sin almacenamiento de código fuente.

## 🚀 Instalación y Uso

### Requisitos
- Node.js (v18 o superior)
- npm o yarn

### Pasos para el despliegue local

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/gitanoxx/frencryptor.git
   cd frencryptor
   ```

2. **Instalar dependencias:**
   ```bash
   npm install
   ```

3. **Iniciar el servidor de desarrollo:**
   ```bash
   npm run dev
   ```

4. **Construir para producción:**
   ```bash
   npm run build
   npm start
   ```

## 🛠️ Tecnologías Utilizadas

- **Frontend:** React 19, Vite, Tailwind CSS, Lucide React, Motion.
- **Backend:** Node.js, Express.
- **Seguridad:** javascript-obfuscator, Base64 Custom Processing.
